# NOHN 340B Internal Audit Checklist

A React/Vite checklist app for structured 340B internal audit decision support.

## PHI warning

Do not enter patient names, DOB, MRN, phone numbers, addresses, or any other identifying information.

Saved audits are stored locally in the browser using audit number only. They are not shared across users or devices.

## Local setup

```bash
npm install
npm run dev
```

## Netlify settings

Build command:

```bash
npm run build
```

Publish directory:

```bash
dist
```
